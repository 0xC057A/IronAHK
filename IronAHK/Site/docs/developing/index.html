Developing

Requirements

Windows:

  • Visual C# Express or professional
  • NUnit 2.4.7
  • msysgit

Unix:

  • MonoDevelop with NUnit add-in
  • Git

Debian and Ubuntu users can install the necessary software by running:

apt-get install monodevelop monodevelop-nunit git-core

Source

The preferred way to obtain the source code is to clone the Git repository which is located at: git://gitorious.org/ironahk/mainline.git

Building

A release build can be created by running make on all operating systems.
To pass all debug tests download the AutoHotkey test suite and extract the files to the Tests/Scripting/Code directory.
